Hướng dẫn triển khai API Versioning với FastAPI
1. Cài đặt môi trường
Chúng ta sẽ sử dụng FastAPI và Uvicorn (một ASGI server nhanh nhất cho Python).

Bash
pip install fastapi uvicorn
2. Cấu trúc thư mục chuyên nghiệp
Trong Python, việc tách biệt các phiên bản bằng Sub-applications hoặc APIRouter là cách làm tốt nhất:

Plaintext
project/
├── main.py          # Điểm khởi chạy ứng dụng
├── api/
│   ├── v1/
│   │   └── endpoints.py
│   └── v2/
│       └── endpoints.py
3. Logic triển khai chi tiết
V1 (Legacy): Sử dụng các Model dữ liệu cũ. Chúng ta sẽ "ép" thêm các Header vào Response để báo hiệu việc sắp đóng cửa API.

V2 (Breaking Change): Thay đổi cấu trúc Model (ví dụ: dùng Pydantic để định nghĩa lại schema dữ liệu).

Lifecycle Management: Sử dụng Middleware hoặc Dependencies để tự động chèn các thông tin Sunset (ngày tắt) và Deprecation.
